Preguntas

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Semejanzas

  • Tienen una sintaxis parecida
  • Comparten nombres de clases como Math o Date

Diferencias

  • JavaScript es un lenguaje interpretado mientras que Java es compilado
  • JavaScript es débilmente tipado y Java es fuertemente tipado
  • Java es recomendado para back-end y JavaScript para front-end
  • Java usualmente utiliza más recursos

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

  1. now(): regresa fecha actual
  2. getDate(): regresa el día del mes
  3. getFullYear(): regresa el año actual
  4. getMonth(): regresa el mes actual
  5. getHours(): regresa la hora actual

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

  1. push(): para insertar elemento
  2. length(): regresa la longitud del arreglo
  3. join(): regresa un string con todos los elementos
  4. indexOf(): regresa el indice del elemento buscado
  5. slice(): regresa una copia de todo o parte de un arreglos?

¿Cómo se declara una variable con alcance local dentro de una función?

Las variables locales de una función deben de declararse dentro de la función ya sea con var, let o const.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Usar variables locales dentro de una función implica que solo van a poder ser usadas dentro de esta función aunque sean globales.
